## About

A task management application featuring animated circular progress indicators. The app displays two dynamic circles - one tracking completed tasks and another for unfinished tasks - both connected to and updated by your task list in real-time.

## Built With

- HTML
- CSS
- Vanilla JavaScript (no frameworks or libraries)

## How to Run

Simply open `index.html` in your browser.
